Home Screen

The home screen serves as the central hub, featuring a dynamic display of the current time (hours, minutes, and seconds), date, and day. This interactive element provides users with real-time information as they navigate through the project.

Calculator Page

The Calculator Page is a practical tool for performing basic mathematical operations. Users can easily execute calculations for:

  • Addition
  • Subtraction
  • Multiplication
  • Division

Singers Page

The Singers Page allows users to experience the fusion of music coding and vocal performance. This page features background music created using Scratch, complemented by added vocals for the song "Roja Janeman." It provides a unique opportunity for users to explore and appreciate the art of music composition and vocal integration.

Learning Page

The Learning Page offers animated explanations of key concepts in Operating Systems and Python including:

  • First-Come-First-Serve (FCFS)
  • Shortest Job First (SJF)
  • Priority Scheduling
  • 'for' loop in Python

Kids Page

The Kids Page is designed to captivate young learners with animated nursery rhymes. The collection includes popular rhymes such as:

  • Ba Ba Black Sheep
  • Jack and Jill
  • Johny Johny Yes Papa
  • Mary Had a Little Lamb
  • Twinkle Twinkle Little Star
  • Humpty Dumpty

Game Page

The Game Page features three exciting games, each with unique challenges and rewards:

  • Box Hop: Players jump over obstacles, and if they fail, a meme of laughing is displayed.
  • Floppy Bat: Players navigate a bat to avoid pillars, with a laughing meme displayed upon failure.
  • Green Zone Racer: Players race within a zone, with boosters available, and fails if they go off the road.

Video Page

The Video Page showcases animated stories created through sprites and code blocks. Featured stories include:

  • Cinderella
  • Hamlet
